2017 NFL Season Data Analysis in R

Date

May 2019

Description

This was a project for Math 23c,  a course entitled "Mathematics for Computation, Statistics, and Data Science."

The project uses numerous computational and statistical methods, as well as simulations, to rigorously analyze play-by-play data from the 2017 NFL season in R.
