Intertemporal Dynamics of Cryptocurrency Economies

Date

May 2019

Description

This was my final project for CS 236r,  a course entitled "Topics at the Interface Between Economics and Computation," which this year focused on blockchain and cryptocurrenices. 

The project uses economic models to address the optimal "monetary policy" (for example, inflation or deflation rate) for a cryptocurrency in a single-currency environment. This analysis is applied intertemporally in a small, closed setting to observe endogenous behavior.
